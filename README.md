# 🚚 Utkarsh Enterprises - Backend

A Spring Boot backend for the **Utkarsh Enterprises Courier Management System**. It provides REST APIs for managing courier bookings, customers, products, shipment tracking, proof of delivery, and reports.

---

## 🚀 Features

- 📦 Booking Management API
- 👥 Customer Management API
- 📦 Product Management API
- 🔍 Parcel Tracking API
- 📝 Proof of Delivery (POD)
- ✍️ Signature Upload
- 🖼️ Delivery Image Upload
- 📊 Reports API
- 🗄️ MySQL Database Integration
- 🌐 RESTful APIs

---

## 🛠️ Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Railway

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Raiki-rani/utkarsh-enterprises.git
```

### Configure Database

Update the MySQL connection in:

```
src/main/resources/application.properties
```

### Run the Project

```bash
./mvnw spring-boot:run
```

or

```bash
mvn spring-boot:run
```
